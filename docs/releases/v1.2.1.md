# QwenVoice 1.2.1

QwenVoice 1.2.1 is a polish and packaging follow-up release focused on light-mode quality, cleaner shipped behavior, and more reliable packaged builds.

- Improved the light-theme studio UI so Custom Voice, Voice Design, and Voice Cloning feel more balanced and consistent.
- Refreshed the project page visuals to match the current app and removed outdated product framing.
- Simplified shipped voice-cloning behavior by removing unsupported tone-delivery plumbing that was not meant to be exposed.
- Tightened automated appearance validation with committed light and dark baselines plus forced-appearance design checks.
- Kept packaged builds strict about bundled runtime dependencies, including Python, ffmpeg, and runtime-manifest verification.
