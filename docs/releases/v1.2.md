QwenVoice 1.2 focuses on a smoother studio workflow, more reliable playback, and stronger packaged builds.

- Improved the Studio interface across Custom Voice, Voice Design, and Voice Cloning with cleaner layouts and smoother navigation.
- Improved streaming preview and playback reliability, including fixes for duplicate playback, short-clip behavior, and live preview timing.
- Added the option to save designed voices to Saved Voices so you can reuse them later in Voice Cloning.
- Improved the voice-cloning workflow with reusable saved voices and more dependable generation handling.
- Strengthened the packaged macOS 15 and macOS 26 builds so the app ships with the runtime dependencies it needs to run reliably.
- Improved overall stability across launch, playback, and packaged app validation.
