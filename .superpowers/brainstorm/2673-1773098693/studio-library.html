New Features in This Redesign

History Additions

  • Sort dropdown (Newest, Oldest, Duration, Mode)
  • Table layout with columns for metadata
  • Mode badges with color coding
  • Missing file indicator (⚠ icon + dimmed row)
  • Inline play on row click
  • Delete confirmation dialog

Voices Additions

  • Edit button (✎) for rename + transcript update
  • Inline play button per voice
  • Source filename shown
  • Delete with confirmation dialog
  • Transcript status badge (has/no)
