A. Stacked Toolbar Rows

Same pattern as Custom Voice but with more rows. Row 1: saved voice chips + import button. Row 2: transcript field. Row 3: tone chips + speed slider. Consistent left-aligned labels (Source, Transcript, Tone). Everything stays above the text area.

B. Collapsible Reference Panel

Reference voice setup lives in a collapsible card at the top. Shows the active voice with status details, other voices as chips, transcript inline. Collapses to a single line showing "🎤 Sarah Clone · Has transcript" once configured. Tone and speed below as flat chips.
